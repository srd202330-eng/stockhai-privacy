Introduction
This Privacy Policy describes how Viv Tech ("we", "our", or "us") collects, uses, and shares your personal information when you use the StockHai mobile application ("App"). By using StockHai, you agree to the collection and use of information in accordance with this policy.

Information We Collect
1. Account Information
Your name and email address

Mobile number for verification

Profile picture (optional)

2. Order and Chat Data
Order details (product information, quantities, prices)

Chat messages between buyers and sellers

Timestamps and chat participants

Attachments shared in chats (images, documents)

3. Device Information
Device model and operating system

Unique device identifiers

IP address and network information

4. Usage Data
App interaction patterns

Feature usage statistics

Error logs and crash reports

How We Use Your Information
For App Functionality:
Order Management: Create, track, and manage orders

Chat System: Enable communication between different order parties

User Authentication: Verify and secure your account

Notifications: Send order updates and chat messages

For Improvement:
Analyze usage patterns to improve features

Fix bugs and technical issues

Develop new features based on user needs

For Communication:
Send important service updates

Respond to your queries and support requests

Notify about order status changes

Permissions Required and Why
1. Internet Access
Required for syncing orders and chats in real-time

Enables push notifications for new messages

2. Storage/Photos/Media Access
Allows uploading product images in chats

Enables sharing documents related to orders

Used for profile picture upload

3. Camera Access
Take pictures of products directly in chats

Scan documents or barcodes

4. Notifications
Alert you about new chat messages

Notify about order updates

Chat System Specific Information
How Chats Work:
Each order has a separate chat thread

Only order participants can access specific chat

Chats are stored securely on our servers

Chat history is preserved for order reference

Chat Data Handling:
Messages are encrypted in transit

Chat data is stored as long as the order exists

You can delete individual messages (if feature available)

Export chat history option available
